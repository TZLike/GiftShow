
###GiftShow(注意：仅供娱乐，不涉及任何商业信息)
===
**采用Charles抓取数据**

* Swift3.0
* 适配苹果现有机型
* 兼容iOS8.0以上系统
* 采用Autolayout+Xib布局
* 三方库使用：  SDWebImage (图片加载) 、AFNetworking（网络请求） 、MJRefresh、SnapKit（布局）
